这个页面不是操作台，而是一份面向课堂讲解的系统说明。它解释这套像素 GeoAI 实验室里，什么是智能体、什么是 GeoAI、为什么经济约束是底层驱动，以及市场、企业竞争、游客需求层、微博、政府、主线新闻、地下赌场和 LLM prompt 工程怎样共同长出复杂行为。

多智能体 GeoAI 行为涌现 经济底层机制 制度与公共服务 企业竞争 游客观察 地下赌场

1. 什么是智能体

从 NPC 到决策者

在这个系统里，智能体不是一段固定台词，而是一个有状态、有记忆、有目标、能感知环境并作出行动的计算实体。

  • 身份层：名字、角色、专长、说话风格
  • 状态层：心情、压力、体力、专注、满意度
  • 社会层：关系、盟友、对手、最近互动
  • 经济层：现金、存款、信用、股票、借贷、房产
  • 记忆层：短期记忆、长期挂念、memory stream、即时意图

2. 人格化是怎么做出来的

像人不是自动出现的

系统里的“人格化”不是给每个人写一串固定台词，而是把身份、口吻、欲望、关系和公开形象叠在一起。这样同一个人私下说话和公开发帖会不一样，紧张时和放松时也会不一样。

  • 身份钩子：林澈、米遥、周铖、芮宁、凯川各有不同的私聊和公开人格
  • 场景钩子：私聊、微博原发帖、微博回复、政府公告分开写
  • 情绪钩子：压力高、满意高、资金紧时，说法会变
  • 输出清洗：去掉黑话、口头禅和内部代号，让表达更像日常中文
人格化 = 身份设定 + 场景口吻 + 情绪状态 + 中文清洗

3. 记忆系统怎么分层

短期记忆 / 长期挂念 / Memory Stream

记忆系统不是一个框里堆所有旧句子，而是按时间尺度分层。短期记忆记录刚发生的事，长期挂念保留跨天还忘不掉的重点，Memory Stream 则把“眼下最有用的几条”压缩出来，供角色当前行动使用。

  • 短期记忆：最近看到、听到、亏了、赢了、吵了、借了的钱
  • 长期挂念：跨天还在心里的市场趋势、人际分歧和外部事件
  • Memory Stream：当前时刻最该被想起的几条线索
  • 去重与收敛：连续几天类似的大盘记忆会合成一条趋势挂念
事件 -> 短期记忆 -> 长期挂念压缩 -> 当前行动时的 Memory Stream

4. 什么是 GeoAI

空间智能主线

GeoAI 是地理空间人工智能。它强调问题和数据带有空间位置、区域结构、流动关系和地图语境。

在这个项目里，GeoAI 既是研究主题，也是一个会影响任务、口碑、新闻、市场和角色行为的主线变量。

GeoAI Progress -> Research Milestone -> News / Reputation / Market Impact

5. 系统为什么会涌现

局部规则 -> 整体现象

系统并没有直接写死“谁会变富”“谁会结盟”“谁会走向灰产”，而是通过很多局部规则相互作用，自然形成长期结果。

  • 现金低于阈值会去打工
  • 信用差会提高借贷成本
  • 游客会带来外部消息和消费
  • 监管和税率会改变行为成本
  • 市场涨跌会反过来影响情绪和风险偏好
  • 公开舆论会放大消费、游客和制度判断

这些局部规则叠加后，就会长出资产分化、关系极化、联盟、冲突和制度反应。

6. 为什么经济规律是底层机制

稀缺性驱动系统

表面上这个世界里最显眼的是地图、对话和关系，但真正持续驱动行为的是经济约束。

  • 没钱要打工
  • 有存款会缓解风险
  • 高负债会改变聊天和决策
  • 房产和股票会改变财富轨迹
  • 税收和保障会改变制度反馈
行为 = f(欲望, 关系, 记忆, 现金压力, 市场信号, 制度成本)

7. 系统之间怎么运行

世界循环
空间系统
地图、房屋、公司、市场、旅馆、政府设施决定相遇概率和行为发生地点。
时间系统
日夜、天气和工作日推进决定打工、休息、游客到访和每日结算。
社会系统
对话、记忆、关系、联盟、冲突和和解链决定谁愿意合作、借钱或对抗。
经济系统
股票、银行借贷、存款、消费、房产、工资和通胀形成持续压力。
制度系统
税率、监管、保障、财政结算、公共服务和政府投资提供制度性反馈。
企业竞争系统
旅馆、集市、工坊、合作社和后街商行会围绕价格、品质、客流、微博舆情和灰色引流互相抢客。
游客观察系统
游客不再只是背景流量，而是被单独观察的需求层：他们会消费、投资、赌博、看房、发微博，并带着外部消息改变小镇。
开放系统
游客、Brave 新闻或系统新闻、节庆活动和随机事件把外部冲击带进小镇。
公开舆论系统
小镇微博把私人想法变成公开信号，再把热度、可信度和围观传播回市场、游客、政府和关系网络。
灰色经济系统
后巷地下赌场把私人赌局、赌税、公开炫耀、灰案注册和监管抽查看成同一个反馈回路。

8. 小镇微博为什么重要

公开信号层

小镇微博不是“附带的聊天页”，而是系统里的半公开舆论场。玩家、核心智能体、游客和政府都可以发帖、回帖、引用、点赞、转发和围观。现在还可以按情绪、游客投资、政府回应和赌场传闻做筛选，并用阅读锁定停住当前时间线。

  • 私人体验会被公开化，变成大家都能看到的信号
  • 高热帖子会进入记忆、晨报和地图围观反应
  • 游客热帖会推高到访与消费预期
  • 政策帖会进入政府已知信息，改变建设和监管判断
  • 市场帖会改变情绪和交易倾向，乐观叙事还会反推消费欲望
公开帖子 -> 热度 / 可信度 -> 记忆 / 市场 / 游客 / 政府 -> 下一轮行为

10. 为什么要做 Prompt 工程

像人话不是自动出现的

这套系统后来专门做了一轮 prompt 调优，因为“模型能生成中文”不等于“角色说的话就像人”。系统现在会把私聊、微博原发帖、微博回复和政府公告拆成不同模板；Qwen 和 GPT 也按不同模型习性分别收口。

  • 私聊更像当面接话，公开微博更像公开表态
  • 政府账号单独走正式公告与正式回应模板
  • 内部代号先翻译成中文地名，例如 data_wall -> 果园坡地
  • 长期记忆会先压成更少、更稳的“长期挂念”
  • 对话卡里的 话题 / 要点 / 展开详情 现在优先都从真实 transcript 抽取，不再一边写游客默认关注点、一边展示另一段原话
像人话 = 任务模板 + 角色口吻 + 中文清洗 + 记忆收敛 + 记录对齐

11. 地下赌场为什么值得观察

灰色经济节点

地下赌场不是单独小游戏，而是一个把冒险冲动、现金波动、公开炫耀、赌税和监管绑在一起的灰色经济节点。它现在更像情绪与舆论放大器，而不是单纯把全镇现金吸走的黑洞。

  • 玩家走到门口后可以手动触发赌局
  • 智能体和游客会在资金紧张、资金很多或情绪高涨时低频自主赌博
  • 赢大钱会引发炫耀、消费冲动和微博传播，输钱则可能导致争执、压力和资金紧迫
  • 赌税进入财政，赌场热度又会提高灰案和监管抽查风险
  • 市场页已经能直接看最近 10 笔赌局和最大输赢榜
  • 最近一轮调优刻意降低了赌场抽走全镇现金的能力，转而强化它对人物情绪、微博热帖和灰案的放大作用
赌局 -> 现金波动 / 情绪变化 -> 微博扩散 -> 赌税 / 灰案 / 抽查

12. 企业竞争为什么会自己长出来

利润 / 客流 / 舆情

企业竞争不是只改价格。现在小镇里的企业会同时追求利润最大化、吸引更多客流，并尽量减少负面舆情带来的损害。

  • 企业会按价格、品质、目标客群和策略争抢居民与游客
  • 玩家、智能体和游客会在微博讨论企业的质量、态度、员工状态和灰色路数
  • 企业自己也会发帖回应，决定是压价、提质、解释还是继续硬抢客流
  • 正规企业更像营业说明和服务公告；灰色企业更会包装、更会强调便宜，但也不能直接威胁顾客
  • 后街商行这类灰色企业，还会把抢客、压价和灰色引流带进灰案与监管链
  • 如果企业公开话术太冲，口碑、可信度和利润都会一起掉，这会直接影响它能不能活下去
企业竞争 = 价格 / 品质 / 客流 / 舆论 / 监管 的同时博弈

13. 游客为什么需要单独观察

外来需求层

游客现在不只是“来消费的人”，而是小镇里的外来需求层。他们会带来现金流、微博热度、股票试水、赌场热度和房产兴趣。如果把这些都继续塞在市场页边角里，学生很难看出游客到底在改变什么。

  • 游客会消费，也会试水投资、轻量卖出、赌博和看房
  • 游客会把外部消息和情绪带进小镇
  • 游客会在微博上放大“值不值得来、值不值得买、这地方靠不靠谱”
  • 系统现在采用 `总游客池 10 / 活跃游客 7` 的分层机制，让世界更热闹，但不把地图和日志噪声打爆
  • 游客名字现在优先使用自然中文名，旧状态里的数字后缀会自动清洗，避免游客像“临时编号对象”而不是外来人口
游客层 = 消费 + 投资 + 赌博 + 看房 + 微博 + 外部消息

14. 灰色交易现在怎么分

地下行为细分

灰色交易不再只是模糊的“私下交换”。现在系统会把几类更具体的地下行为拆出来，方便观察它们怎样进入微博、灰案和监管链。

  • 房产私下转手：不走正式挂牌，私下交钥匙、租约或空屋
  • 账外回扣：围绕派单、活口或资源倾斜形成的回扣
  • 私货过手：来路不明的货箱、假货或临时过手的私货
  • 拉高出货：盘前串联、先顶后砸的灰盘操作
私下交换 -> 灰案 -> 微博 / 市场 / 监管 -> 下一轮冲突

15. 最近的系统平衡怎么调

机制优化

最近一轮调优不是加新按钮，而是在调系统的“力学平衡”，避免所有财富都被罚款或赌场吸走。

  • 降低罚款触发率和罚款权重，减轻财政对惩罚收入的依赖
  • 增强正向游客消费、市场热度和生活满意度之间的正反馈
  • 减少重复社会热点播种，避免微博像机械复读
  • 让赌场更多影响个体情绪、微博和关系，而不是只抽走现金

  3. 去小镇微博看公共舆论、热帖、传播链和政府回应。
  4. 去财政页看税率、保障、政府投资和制度反馈。
  5. 最后到日志页，把实时对话、事件流、晨报和新闻时间线串起来。
