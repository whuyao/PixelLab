# GeoAI Pixel Lab 长时程模拟复盘

截至第 312 天午间状态

## 1. 文档目的

这份文档复盘当前 `LocalFarmer / GeoAI Pixel Lab` 沙盒在前 300 多天中的系统演化过程，重点回答两个问题：

1. 系统本身在长时程运行中，哪些机制逐步成形并开始主导行为。
2. 五个智能体在这 300 多天里，各自朝什么方向“进化”。

这份复盘不是单纯看当前快照，而是结合了日级快照和活动日志。

## 2. 数据来源与方法

分析基于两类数据：

- SQLite 快照库 [save/localfarmer.db](/Volumes/Yaoy/project/LocalFarmer/save/localfarmer.db)

## 3. 一个关键解释：这不是“单版本 312 天”

必须先说明一件事：

这 312 天不是“同一套规则从第 1 天一直跑到第 312 天”，而是一个持续增量升级的模拟世界。也就是说：

- 前期先有社交、任务、新闻、GeoAI 推理
- 中期加入市场、借贷、灰色交易、Lab Daily
- 后期再加入地产、消费、银行、公司打工、日常开销

因此，这份文档里的“系统升级”既包括：

- 模拟内部自然形成的经济与关系结构
- 也包括规则层在运行中被逐步接入后，对世界行为造成的结构性改变

最明显的例子就是：

- 房地产持有从第 `234` 天开始明显出现
- 公司打工系统从第 `307` 天才开始在快照里留下连续痕迹

所以，后 80 天和前 80 天并不能视为同一规则集下的可直接对照实验，而应视为“长时程世界 + 连续制度升级”的混合过程。

## 4. 总体结论

从第 1 天到第 312 天，系统从“高波动社交实验室”演化成了一个明显金融化、资产化、关系极化的复杂生态：

- `GeoAI 进度` 从 `30` 增长到 `1315`
- 实验室 `口碑` 从 `0` 修复到 `100`
- 团队现金从 `495` 增长到 `51281`
- 市场长期偏牛，但中途经历了多轮 `风险市`
- 角色对玩家的关系最终几乎全部拉满，但角色彼此之间出现了明显极化
- 房地产、借贷、灰市和公司工资共同把经济层做厚了

如果用一句话概括：

这个系统已经从“几个人在实验室里聊天”演化成了“一个带科研主线、资本积累、灰市暗线、资产配置和关系政治的长期社会经济模拟器”。

## 5. 系统演化阶段

### 阶段 A：启动混乱期（第 1-4 天）

这一阶段的特点是：

- 口碑几乎归零
- 玩家现金极低，启动态接近破产
- 平均压力高，个体信用与心情波动大
- 系统还没有形成稳态的财富、地产或劳动闭环

第 1 天日末的日级快照显示：

- 玩家现金只有 `$7`
- 玩家信用为 `0`
- 实验室口碑为 `0`
- 市场指数只有 `97.82`
- 市场情绪为 `-32`

也就是说，系统一开始不是“顺滑繁荣”，而是从近乎失序的边缘状态起步。

### 阶段 B：快速修复与研究突破期（第 5-22 天）

这一阶段是系统第一次进入正反馈：

- 口碑在第 `5` 天就重新站上 `50` 和 `80`
- `GeoAI 进度` 在第 `12` 天突破 `500`
- 到第 `22` 天，`GeoAI 进度` 已破 `1000`

这说明早期最强的驱动器不是经济，而是：

- 对话
- 新闻注入
- 研究推进
- 任务奖励

换句话说，系统先完成的是“认知-关系-研究”的正反馈闭环，而不是“金融-资产”的闭环。

### 阶段 C：社交与市场耦合期（第 23-125 天）

这阶段系统开始出现更稳定的多层耦合：

- 社交线程和持续对话变多
- 外部新闻与市场波动开始联动
- 角色的资金、信用、关系不再各自独立，而是相互影响

这说明这个世界在中期最重要的特征已经不是“任务驱动”，而是“关系驱动 + 外部信息驱动”。

### 阶段 D：资本积累期（第 126-233 天）

这是一个非常重要的制度变点。

第 `234` 天开始：

- 钱不再只通过股票和借贷流动
- 房地产成为稳定现金流和满意度调节器
- 经济活动开始具备空间锚点

### 阶段 F：劳动制度化期（第 307-312 天）

这是最近才接入、但影响已经非常明显的层。

第 `307` 天开始，公司打工正式进入经济回路：

- 公司：`青松数据服务`
- 现金低于 `$50` 会强烈触发打工
- 工资受精力投入影响
- 每天也会有生活开销

截至第 312 天：

- 累计发薪：`$8881`
- 工作场次：`177`

这一步的意义不是“多一个挣钱方法”，而是把经济系统闭环补全了：

- 现金不够 -> 去打工
- 有钱后 -> 消费 / 借贷 / 买房 / 入市
- 高负担时 -> 继续劳动或转向灰市套利

这使系统进入了真正的“劳动-消费-投资-信用”循环。

## 10. 当前系统暴露出的结构特征

### 10.1 玩家关系过强，同行关系过度极化

最终几乎所有角色都对玩家 `100`，但同行之间大量掉到 `-100`，这会导致：

- 玩家像“全员共同情感锚点”
- 同行网络却越来越像阵营对立

这会让系统后期更像“围绕玩家的政治经济体”，而不是一个真正自主平衡的社群。

### 10.2 满意度系统后期趋于饱和

后期几乎所有角色满意度都接近 `100`，说明：

- 生活面机制已经补全
- 但上限饱和后，区分度开始下降

后续可以考虑引入：

- 满意度衰减
- 品质升级门槛
- 阶层比较效应

### 10.3 财富分化已经非常明显

以第 312 天为例：

- 凯川现金超过 `4 万`
- 米遥、芮宁只有约 `1 千`
- 周铖、林澈在 `4 千` 量级

说明系统已经进入明显的“资本赢家-边缘跟随者”结构。

### 10.4 工作系统接入较晚，但已成为闭环关键

工作系统只在第 `307` 天后才进入长期轨迹，但短时间内就跑出了：

- `177` 次工作
- `8881` 累计工资

这说明它作为保底收入机制是有效的，后续值得继续增强。

## 11. 下一步最值得继续研究的方向

基于这 312 天的结果，下一步最有价值的不是再加零散功能，而是收下面 4 个结构问题：

### 11.1 收玩家中心化

当前玩家和所有角色关系最终都极高，建议加入：

- 观察模式下关系推进衰减
- 玩家干预成本
- 玩家信誉和口碑分离

### 11.2 收同行极化

目前同行关系很容易一路打到极端，建议加入：

- 自然回弹
- 冷却期
- 和解事件链

### 11.3 让财富赢家承担更高外部风险

现在财富越高越容易继续滚雪球，后续可加：

- 更强税费 / 维护费
- 暴露风险
- 监管抽查
- 资产折价周期

### 11.4 让工作系统和灰市系统更深耦合

这一步已经很有潜力，可以继续做：

- 公司内部派单不均
- 回扣链
- 工资拖欠
- 借工资名义洗钱
- 通过劳动换内幕或关系

## 12. 总结

这 312 天最重要的结论不是“谁赚了多少钱”，而是：

系统已经自然长成了一个多层耦合结构。

它现在同时拥有：

- 研究推进
- 社交关系
- 市场投机
- 银行信用
- 人际借贷
- 灰色交易
- 房地产
- 劳动供给
- 每日生活消耗

五个智能体也不再只是“几种人设模板”，而是已经在这个结构里分化成了不同位置：

- 林澈：高压积累者
- 米遥：低信用边缘机会主义者
- 周铖：制度内稳定赢家
- 芮宁：被透支的情绪型孤岛
- 凯川：最成功的资本化赢家

如果继续跑下去，这个系统最值得观察的，将不再是“会不会继续涨”，而是：

- 会不会出现阶层固化
- 会不会形成长期政治联盟
- 会不会在高财富与高灰市活动下爆发一次真正的系统性危机
